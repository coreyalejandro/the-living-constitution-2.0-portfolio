## What This Repository Is

The Living Constitution 2.0 is a local-first governance/control-plane repository for managing C-RSP contracts, module registries, artifact registries, route registries, evidence records, verification outputs, research-to-paper-to-product templates, portfolio-safe public data, and project truth status.

It exists to prevent prototypes, drafts, partial implementations, and verified systems from being collapsed into one misleading public claim.

## C-RSP Is a Subsystem

C-RSP means **Constitutionally-Regulated Single Pass**.

C-RSP contracts are a core governance mechanism inside The Living Constitution 2.0, but they are not the whole system.
